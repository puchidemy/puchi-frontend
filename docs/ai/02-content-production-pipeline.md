# AI Content Production Pipeline

Idea
→ Research
→ Story Draft
→ Story Validation
→ Linguistic Analysis
→ Vocabulary Extraction
→ Grammar Extraction
→ Activity Generation
→ Media Generation
→ QA
→ Human Review (optional)
→ Publish

## Responsibilities

Story Writer:
- Produce engaging story.

Validator:
- CEFR
- readability
- repetition
- cultural accuracy

Generator:
- exercises
- audio
- artwork prompts

Publisher:
- versioning
- analytics
